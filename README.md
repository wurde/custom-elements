## Development

```bash
# Run a local development server.
npm start

# To run a local Storybook instance.
npm run storybook

# Linting with ESLint, Prettier, and Types.
npm run lint
# Fix linting errors.
npm run format

# Run tests in watch mode.
npm run test:watch
```
